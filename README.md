# بوت مراقبة منصة أضاحي (adhahi.dz) - النسخة الثانية

هذا البوت يقوم بمراقبة الموقع الرسمي الجزائري لحجز الأضاحي `adhahi.dz` باستخدام تقنية Playwright للتعامل مع المحتوى الديناميكي، ويرسل تنبيهات فورية عبر تلغرام عند توفر أي ولاية من الولايات الـ 58 للتسجيل.

## التحديثات في النسخة الثانية (V2)
- **دعم المحتوى الديناميكي**: استخدام Playwright لمحاكاة متصفح حقيقي وقراءة القائمة التي تظهر بعد البحث.
- **مراقبة دقيقة**: الكود يكتشف الولايات المتوفرة (باللون الأخضر) وغير المتوفرة (باللون الوردي) بناءً على النصوص الظاهرة في الصفحة.
- **تنبيهات ذكية**: يرسل تنبيهاً عند فتح الولاية وعند إغلاقها.
- **الحالة الأولية**: عند أول تشغيل، يقوم بحفظ الحالة الحالية دون إرسال تنبيهات مزعجة لكل الولايات المفتوحة مسبقاً.

## المتطلبات
- Python 3.8+
- Playwright (يتم تثبيته تلقائياً عبر Procfile)

## النشر على Railway.app
1. ارفع الملفات إلى مستودع GitHub.
2. في Railway، أنشئ مشروعاً جديداً من المستودع.
3. أضف المتغيرات:
   - `BOT_TOKEN`: توكن البوت من BotFather.
   - `CHAT_ID`: معرف الدردشة الخاص بك.
4. سيتكفل ملف `Procfile` بتثبيت متصفح Chromium والاعتمادات اللازمة قبل تشغيل البوت.

## الأوامر
- `/start`: بدء البوت وعرض المعلومات.
- `/status`: عرض الولايات المتوفرة حالياً فقط.
- `/list`: عرض قائمة بجميع الولايات الـ 58 وحالة كل منها.

## ملاحظة تقنية
تم تصميم البوت ليكون مرناً؛ فإذا فشل Playwright لأي سبب، سيحاول البوت استخدام طلبات API مباشرة (في حال توفرها) لضمان عدم توقف المراقبة.
